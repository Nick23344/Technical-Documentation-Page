Introduction

At the core of programming lie essential concepts that empower developers to build dynamic and functional software.
Variables act as data containers, storing a range of information that can be manipulated as needed. Declaring variables involves specifying their names and data types, allowing the computer to allocate memory appropriately. Conditionals introduce decision-making, enabling code to respond intelligently to varying scenarios based on true or false conditions. Loops, on the other hand, enable the repetitive execution of code blocks, streamlining tasks like iterating through lists or performing calculations.
These concepts together form the building blocks of programming, enabling the creation of adaptable, interactive, and efficient applications.

What you should already know

This guide assumes you have the following basic background:

  • Moderate experience in the use of computers.
  • Some or minimum experience in programming.
  • General understanding on the differences of coding languages.

Variables

Variables are used to store and manage data in a computer's memory. They act as placeholders or containers for values that can be manipulated, accessed, and updated during the program's execution.

Think of a variable as a named storage location that can hold different types of data, such as numbers, text, or complex structures. By using variables, programmers can create dynamic, adaptable, and interactive code.

Declaring variables

Variables are declared with a name and a data type to specify the kind of value they will hold. They can also be initialized with an initial value. For example:

int age = 25 // an integer variable
String name = "John" // a string variable
float pi = 3.14 // a floating-point variable
boolean is_student = True // a boolean variable

Variables can be assigned new values after their declaration. For example:

age = 30 // assigning a new value to the 'age' variable

Conditionals

Conditionals are used to introduce decision-making into code. They allow the program to execute different sets of instructions based on whether certain conditions are met. In programming, conditions are typically expressed as Boolean expressions, which evaluate to either true or false. Conditional statements, like "if," "else if," and "else" in languages such as Python or C, enable programmers to control the flow of their code and make it respond intelligently to different scenarios. For example: (code in Python)

age = 18 if age >= 18:
  print("You are an adult.")
else:
  print("You are not an adult.")

Loops

Loops are constructs that allow a section of code to be executed repeatedly. They are used when you need to perform a certain action multiple times, like iterating through a list of items or performing calculations.
There are two main types of loops: "for" loops and "while" loops.

  • "for" loops are used when you know how many times you want to iterate
  • "while" loops are used when you want to keep iterating as long as a specific condition is true.
For example:

for (Python)

for i in range(5):
  print(i)

while (Java)

int count = 0;
while (count < 5) {
System.out.println(count);
  count++;
}
